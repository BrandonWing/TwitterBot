Anime-Robo
------------------------------------------------------------------

This bot is designed to automatically upload an anime picture every 20 minutes
It uses both a database of links to download pictures from, as well an already
downloaded directory of wallpapers. 

This repository comes with the database already created, but the wallpapers will
have to be downloaded to the users computer using this link:
https://mega.nz/#!O4hUSDAA!kHg_cs59dTYqlc6pkYWIPtTIm63UhV_WxL_SvLh8r2o

The script is made to run on linux.
